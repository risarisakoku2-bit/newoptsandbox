Gravity Sandbox (p5.js)

概要
-----
指を置くと“重力井戸”が現れ、粒子が引き寄せられて渦や軌道を作るインタラクティブ作品です。複数の観客が同時に触ると相互作用が生まれます。

使い方（展示）
----------------
- タッチディスプレイで `index.html` を開く（フルスクリーン推奨）。
- 指で画面を押すと周辺の粒子が引き寄せられます。
- 長押しすると井戸が強くなります。対応端末では押圧（force）も強さに影響します。
- 複数の指で触ると複数の井戸ができ、渦や衝突が生まれます。
- デスクトップでのテスト：マウスを長押しで同様に操作できます。キー1-4で粒子数を切り替えます。

音声について
------------
- このバージョンは触覚に連動した音を生成します（p5.sound を使用）。各タッチに対してトーンが割り当てられ、横位置で周波数、押しの強さ/長押しで音量が変化します。
- 多くのモバイルブラウザは音声再生をユーザー操作までブロックします。最初のタッチがユーザー操作として扱われるため、そのタッチで音が有効になるはずですが、動作しない場合は画面を一度タップしてから試してください。
- パフォーマンスにより音の再生/合成が制限される場合があります（古い端末）。必要なら音をオフにする UI を追加できます。

実行方法
--------
ローカルで簡単に動かすには、ブラウザで `index.html` を直接開くか、簡易HTTPサーバーを使います。

macOS / zsh (推奨、同一ディレクトリで実行):

```bash
python3 -m http.server 8000
# ブラウザで http://localhost:8000/ を開く
```

拡張アイデア
------------
- 粒子数や井戸の最大数をUIで操作できるコントローラを追加
- 井戸の性質（反発、回転力）を増やして多様なダイナミクスを作る
- サウンド（p5.sound）を追加して触覚と聴覚を連動させる
- WebGLシェーダで大量粒子をGPUにオフロードして高密度表現

注意点
------
- 古いタブレットではフレームレートが低下します。粒子数を下げるモードを用意してください。
- ブラウザのマルチタッチ実装差異（identifier/forceの有無）に注意しています。
